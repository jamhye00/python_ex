# 콘솔 음식 주문 프로그램

## 프로그램 설명
사용자는 메뉴를 보면서 음식을 주문할 수 있다.

기능:
1. 메뉴 보기
2. 음식 주문
3. 주문 취소
4. 현재 주문 내역 보기
5. 총 가격 보기
6. 종료

---

## 음식 메뉴

foodMenu = {
    '김치볶음밥': 7000,
    '돈까스': 8000,
    '라면': 4000,
    '제육볶음': 9000
}

---

## 메뉴 출력

1.메뉴보기
2.음식주문
3.주문취소
4.주문내역
5.총금액
6.종료

---

## 기능 설명

1. 메뉴보기
- 음식 이름과 가격 출력

2. 음식주문
- 존재하는 음식이면 주문 성공
- 주문 목록에 저장

3. 주문취소
- 주문한 음식이면 삭제

4. 주문내역
- 현재 주문한 음식 출력

5. 총금액
- 주문한 음식들의 총 가격 계산

6. 종료
- 프로그램 종료
